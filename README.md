# 555 Timer Regulator

## Overview
This project implements a basic voltage regulation and timing circuit using the NE555 integrated circuit.

The goal is to revisit fundamental concepts of analog electronics and document the complete design process.

## Objectives
- Design a NE555-based circuit
- Create schematic and PCB
- Document design decisions and lessons learned

- In development
